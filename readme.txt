the goal of this app to reduce the storage in your device, and to also convert video to pdf, for faster revision.




issues:
1) extracting each frame and comparing taking a long time for even a small video (<5min).
	may have to use GPU for faster processing.

2)	total images are equal to dissimilar images for some videos, even if the images are similar.


HOW TO RUN?

1) keep all the files and folder in the same folder.
2) keep the video in same folder
3) run extract_all_frames_from_video.py
4) then run compare_frames.
5) finally run dissimilar_frames_to_pdf.
